# отчет по проекту: прогнозирование оттока клиентов банка

## общая информация

этот отчет подводит итоги анализа и построения модели по задаче прогнозирования оттока клиентов банка. работа включала этапы разведочного анализа данных, предобработки, моделирования и интерпретации результатов.

---

## 1. характеристика данных

* размер выборки: 10 000 клиентов
* целевая переменная: `exited` (1 - клиент ушел, 0 - остался)
* доля оттока: около 20%
* основные признаки: creditscore, geography, gender, age, balance, numofproducts, isactivemember, estimatedsalary

---

## 2. разведочный анализ данных (eda)

### демографические наблюдения

* возраст: вероятность ухода резко возрастает после 40 лет.
* пол: женщины немного чаще уходят, чем мужчины.
* география: клиенты из Германии уходят чаще всего, из Испании реже всего.

### финансовые показатели

* баланс: клиенты с балансом среднего и высокого уровня чаще покидают банк.
* кредитный рейтинг: низкий рейтинг связан с большей вероятностью оттока.
* количество продуктов: клиенты с одним продуктом чаще уходят, чем те, кто пользуется несколькими.
* активность клиента: неактивные пользователи уходят значительно чаще, это один из сильнейших факторов.

---

## 3. предобработка данных

* категориальные признаки (geography, gender) были закодированы методом one-hot encoding.
* числовые признаки нормализованы для корректной работы моделей.
* данные разделены на обучающую и тестовую выборки с сохранением пропорций классов (stratify).

---

## 4. построение моделей

были протестированы несколько алгоритмов:

* логистическая регрессия
* случайный лес (random forest)
* градиентный бустинг (gradient boosting)

лучшие результаты показал random forest, продемонстрировав:

* accuracy: 86–88%
* roc-auc: около 0.85

---

## 5. анализ важности признаков

наиболее значимыми признаками стали:

* возраст
* признак активности (isactivemember)
* баланс на счете
* географическое расположение (особенно германия)

---

## 6. основные выводы

1. возраст и неактивность являются главными факторы риска оттока.
2. клиенты с одним продуктом или высоким балансом уходят чаще.
3. географический фактор (страна) также играет заметную роль.
4. уровень зарплаты почти не влияет на отток.

---

## 7. рекомендации для бизнеса

1. разрабатывать программы удержания для клиентов старше 40 лет и неактивных пользователей.
2. стимулировать использование нескольких банковских продуктов (кросс-продажи).
3. проводить целевые акции и предложения для клиентов из регионов с высоким уровнем оттока (например, германия).
4. раннее выявление и активация неактивных клиентов с помощью уведомлений и персональных предложений.

---

## 8. заключение

построенная модель позволяет с высокой точностью выявлять клиентов, склонных к уходу, и может быть интегрирована в crm-систему для автоматического прогнозирования риска оттока. это создает основу для разработки стратегий удержания и повышения лояльности клиентов.
